第5章 標準 ID: HQ-0788

問題

情報システム部門の利用説明で、Output Indicatorと「Instruction」の違いを踏まえた説明として最も適切なものを1つ選びなさい。

  1. A. Output Indicatorは出力形式や回答の形を指定する部分であり、「Instruction」はAIに実行してほしい作業や命令を示す部分として区別される。
  2. B. Output Indicatorと「Instruction」は、名称が違うだけで常に同じ意味で使われる。
  3. C. Output IndicatorはAIに実行してほしい作業や命令を示す部分であり、「Instruction」は出力形式や回答の形を指定する部分である。
  4. D. Output Indicatorは第5章では扱われず、試験範囲外の語句である。

解説（正解: A）

正解は1です。Output Indicatorは「出力形式や回答の形を指定する部分」であり、「Instruction」とは役割や対象が異なります。比較問題では、2つの用語の定義を入れ替えた選択肢に注意します。選択肢の前半だけが正しくても、後半で別概念を誤って説明していれば不正解になります。第5章では、プロンプトの構成、実務活用、生成AIの不得意な点を具体的な利用場面で確認します。誤答を読むときは、語句の一部だけで判断せず、説明全体がその用語の定義と対応しているかを確認しましょう。
