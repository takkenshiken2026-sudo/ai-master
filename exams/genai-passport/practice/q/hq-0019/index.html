HQ-0019 · AIの歴史

生成AIパスポートの実践演習模擬問題です。解説付きで個別に学習できます（公式の過去問ではありません）。

第1章 標準 ID: HQ-0019

問題

管理職向けAIリスク研修で出された第三次AIブームに関する説明として、不適切なものを1つ選びなさい。

  1. A. 第三次AIブームは、関連する用語と区別して覚える必要がある。
  2. B. 第三次AIブームは、ビッグデータやディープラーニングの進展によりAI活用が広がった時期。
  3. C. 第三次AIブームを理解する際は、AIの歴史との関係も確認するとよい。
  4. D. 第三次AIブームは、知識を使うエキスパートシステムが注目された時期。

解説（正解: D）

正解は4です。選択肢「第三次AIブームは、知識を使うエキスパートシステムが注目された時期。」は、第三次AIブームではなく別の概念の説明です。第三次AIブームの要点は「ビッグデータやディープラーニングの進展によりAI活用が広がった時期」であり、不適切選択では別分野の説明が混ざっていないかを確認します。正しい説明が複数あるように見える場合でも、主語と説明の対応がずれている選択肢を探すと判断しやすくなります。第1章では、AIの基礎用語を似た概念と区別できるかが重要です。誤答を読むときは、語句の一部だけで判断せず、説明全体がその用語の定義と対応しているかを確認しましょう。
