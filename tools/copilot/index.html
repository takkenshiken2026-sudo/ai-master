Microsoft Copilot（マイクロソフト・コパイロット）は、Microsoftが提供する生成AI支援のサービス群です。ブラウザのチャットから、Word・Excel・Teams内のAIまで幅広く使われます。本記事は全機能の網羅ではなく、試験での定義（MicrosoftとCopilotの対応、GitHub Copilotとの違い）と、Microsoft 365ユーザーにとっての位置づけに絞って書いています。料金やプラン名は2026年6月時点の情報です。契約前は料金表と公式の最新情報を確認してください。

試験で問われる見方

生成AIパスポートでは、Copilotは「Microsoftが提供する生成AI支援機能・サービス群」として覚えるのが基本です。ChatGPT、Gemini、Claudeと並ぶ主要例のひとつで、開発元を入れ替えた選択肢がよく出ます。

よくある誤解は3つあります。「Copilot＝GitHub Copilotだけ」と狭く捉えること（チャットやOffice連携も含む総称です）。「Copilot＝要配慮個人情報」のような法律用語と混同すること。そして「AIが出力を自動的に保証する」と考えること。試験のケース問題では、定義に加えて出力の確認義務や機密情報の入力禁止までセットで問われます。

Microsoft Copilotとは

Microsoft Copilotは、2023年頃から本格展開されたMicrosoft製のAIアシスタントブランドです。個人向けのWebチャット（copilot.microsoft.com）、Windowsに組み込まれたCopilot、Microsoft 365アプリ内のCopilotなど、入口は複数あります。背後ではOpenAIなどのモデルも使われますが、試験では「どのモデルか」よりMicrosoftのサービスとして何をするかが重要です。

実務で効いてくるのはMicrosoft 365との一体感です。Wordで下書き、Excelで数式の説明、Outlookでメール整理、Teamsで会議要約——すでにMicrosoftで仕事をしている組織ほど、導入の意味が大きくなります。逆にGoogle Workspace中心の会社ではGeminiのほうが自然な場面も多いです。

名称が似たGitHub Copilot（コード補完）は別製品です。開発者向けツールとして一覧にも載っていますが、本記事のMicrosoft Copilotとは用途が異なるため、試験では定義を混ぜないよう注意してください。

Copilotでできること（主な機能）

「Copilot」とひと言で言っても、利用場所によってできることが変わります。

Webチャット

copilot.microsoft.comから質問・要約・文案作成。Bing検索と連携し、比較的新しい話題にも触れやすいです。

Microsoft 365連携

Word・Excel・PowerPoint・Outlook・Teams内で文章生成や要約（Microsoft 365 Copilotライセンスが必要な機能あり）。

画像の理解・生成

画像の説明や画像生成系の機能（プラン・地域により異なる）。

Windows統合

Windows 11などに組み込まれたCopilotから、PC操作や質問への回答を行えます。

エンタープライズ機能

組織向けの管理、データ保護、Microsoft Graph経由の社内情報参照（設定・プラン依存）。

モバイルアプリ

iOS・Androidアプリからチャット利用。移動中の簡易質問に向きます。

個人版とMicrosoft 365 Copilotの違い

無料のWeb Copilotで試した体験と、会社で契約したMicrosoft 365 Copilotは、参照できるデータや管理方法が異なります。「家で使った無料版と、会社の環境は同じ」と思い込むと、情報管理の判断を誤りやすいので、業務利用ではIT部門の案内を優先してください。

はじめ方・基本的な使い方

まずは無料のWeb版から試すのが手軽です。

  1. Microsoftアカウントでログインする copilot.microsoft.comにアクセスし、Microsoftアカウントでサインインします。
  2. 質問または依頼を入力する 例：「この文章をビジネスメール向けに短くして」「Excelのピボットテーブルの手順を教えて」
  3. 必要ならファイルや画像を添付する 資料の要約や画像の説明を依頼できます（プランにより上限あり）。
  4. 会社で使う場合はIT部門の案内を確認する Microsoft 365 Copilotは管理者が有効化したうえで、許可された範囲で利用します。

うまく使うためのコツ

  • 利用場所を意識する — Web版・Windows版・Office内ではできることが違う
  • 検索連携の回答も要確認 — 最新情報でも誤りは起こりうる
  • 機密は組織ルールに従う — 無料個人アカウントに社内資料を入れない
  • GitHub Copilotと混同しない — コード補完は別製品・別記事の対象

  • Microsoft 365で日々の業務をしている方 — 既存アプリにAIを載せやすい

よくある質問

Microsoft Copilotは無料で使えますか？

はい。copilot.microsoft.comなどから無料で基本的なチャット機能を利用できます。ただし、Microsoft 365アプリ内のCopilotや企業向け機能は別プラン・別料金です。

Microsoft CopilotとGitHub Copilotの違いは？

Microsoft CopilotはチャットやOffice連携を含む生成AI支援の総称です。GitHub Copilotは開発者向けのコード補完ツールで、名称は似ていますが主な用途が異なります。

Microsoft CopilotとChatGPTの違いは何ですか？

Microsoft CopilotはMicrosoft製で、Word・Excel・TeamsなどMicrosoft 365との連携が強みです。ChatGPTはOpenAI製で、カスタムGPTや多機能な個人向けチャットとして広く使われています。

Microsoft Copilotは何社が提供していますか？

Microsoft（マイクロソフト）が提供する生成AI支援機能・サービス群です。試験では開発元と名称の対応、およびGitHub Copilotなど類似名称との区別が問われることがあります。

会社の業務でMicrosoft Copilotを使っても大丈夫ですか？

個人の無料アカウントに機密情報を入力するのは避けるべきです。業務利用ではMicrosoft 365 CopilotやEnterprise向けプラン、社内ガイドラインの整備が推奨されます。出力の正確性や権利関係は自動保証されないため、利用前に公式の規約を確認してください。
