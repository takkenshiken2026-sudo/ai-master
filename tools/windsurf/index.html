Windsurfとは？機能・料金・使い方をわかりやすく解説

Cursor対抗のAIエディタ。無料枠が充実

無料枠あり Cascadeエージェント
対応環境
  • Windows
  • macOS
  • Linux
更新日：2026年6月7日 読了目安：約7分

Windsurf（ウィンドサーフ）は、Codeium（コーディアム）が提供するAI統合型コードエディタです。VS Codeに似た操作感を持つ独立アプリとして動作し、Cascadeと呼ばれるエージェントが複数ファイルにまたがる改修や調査を支援します。Cursorと並ぶ「AIエディタ」として知られ、無料枠の充実が選ばれる理由のひとつです。本記事は全機能の網羅ではなく、試験で押さえる位置づけ（Codeium製・拡張機能との違い）とCursor・GitHub Copilotとの比較に絞って書いています。料金や機能上限は2026年6月時点の情報です。利用前は料金の考え方と公式の最新情報を確認してください。

試験で問われる見方

生成AIパスポートやG検定では、AIによるコード支援としてChatGPT（Codex系）、GitHub Copilot、Cursorなどが文脈として登場します。WindsurfはCodeiumが提供するAIエディタとして整理します。開発元をCodeiumに入れ替えた選択肢や、「拡張機能」と「単体エディタ」の区別を問う設問に備えておくとよいです。

よくある誤解は3つあります。「Windsurf＝VS Codeの拡張機能だけ」と捉えること（独立したエディタ本体です）。「Windsurf＝Codeium拡張機能と同じ」と同一視すること（拡張機能は既存IDEへのプラグイン、Windsurfは製品そのものです）。そして「GitHub Copilot＝Microsoft Copilot＝Windsurf」のように名称だけで混同すること（提供元と用途が異なります）。

Windsurfとは

Windsurfは2024年頃から本格提供されたCodeium製のAIコードエディタです。見た目やショートカットはVS Codeに近い一方、エディタ本体にAI機能が組み込まれています。単なる補完だけでなく、Cascadeエージェントがプロジェクト全体を読み、複数ファイルの編集・ターミナル操作・Web検索などを組み合わせてタスクを進める点が特徴です。

同じCodeiumブランドでVS Code向けCodeium拡張機能も提供されていますが、Windsurfは別製品です。拡張機能は「今使っているエディタにAI補完を足す」用途、Windsurfは「AI前提で設計されたエディタに乗り換える」用途、と使い分けられます。試験や実務の会話で「Codeium」とだけ言及された場合、文脈によってどちらを指すか確認が必要です。

競合としてよく挙がるのがCursor（Anysphere製）です。どちらもAIエディタ市場の代表格で、エージェント機能や無料枠の設計が似ています。GitHub Copilotは「既存エディタにAIを足す」系の代表で、Windsurfとは製品形態が異なります。

コーディング支援に関わる主な機能を整理します。

Cascadeエージェント

自然言語の指示から、複数ファイルの編集・調査・実行までを段階的に進めるAIエージェント。

インライン補完

入力中に次のコード行や関数を提案。Codeiumのモデルがエディタ内で動作します。

チャット（Cmd+I等）

選択範囲やファイルをコンテキストに、質問・リファクタ・説明をチャットで依頼できます。

VS Code互換の操作感

拡張機能・キーバインド・テーマなど、VS Codeユーザーが移行しやすい設計です。

ターミナル連携

エージェントがコマンド実行を提案・実行し、ビルドやテストの結果を踏まえて修正できます。

コンテキスト理解

開いているファイルやプロジェクト構造を読み取り、より文脈に沿った提案を行います。

AIエディタとの位置づけ

ChatGPTにコードを貼って相談する方式と比べ、Windsurfはローカルのプロジェクトを直接触れる点が強みです。GitHub Copilotのような補完特化型と比べ、エージェントによる自律的な作業を前面に出しています。Cursorと似たカテゴリですが、開発元・無料枠・モデル選定が異なるため、チーム方針に合わせて選びます。

はじめ方・基本的な使い方

デスクトップアプリをインストールし、数分で始められます。

  1. 公式サイトからインストール Windows / macOS / Linux向けインストーラをダウンロードし、セットアップします。
  2. アカウント作成・ログイン Codeiumアカウントでサインインし、無料枠の利用を開始します。
  3. プロジェクトを開く 既存のGitリポジトリまたは新規フォルダを開き、ファイルツリーを確認します。
  4. 補完・チャットを試す コード入力でインライン補完を確認。選択範囲を指定してチャットでリファクタを依頼します。
  5. Cascadeでタスクを依頼 例：「ログイン画面のバリデーションを追加してテストを実行して」と自然言語で指示します。
  6. 差分をレビューしてから適用 エージェントの提案は必ず人の目で確認し、機密情報が含まれていないかチェックします。

Cursor・GitHub Copilotとの比較

名称や用途が似て迷いやすい3つを整理します。

項目 Windsurf Cursor GitHub Copilot
提供元 Codeium Anysphere GitHub / Microsoft
製品形態 独立したAIエディタ 独立したAIエディタ 既存IDE向け拡張・連携

無料で試す ◎（Free枠あり） ○（Free枠あり・上限あり） △（個人向け無料期間等・要確認）

「エディタごとAI前提で使いたい」ならWindsurfかCursor、「今のVS Codeはそのまま補完だけ欲しい」ならGitHub Copilot、という切り分けが現実的です。Microsoft Copilotはチャット・Office連携が中心で、開発特化のGitHub Copilotとは別製品として整理します。

よくある質問

Windsurfは無料で使えますか？

はい。無料プランがあり、AI補完やCascadeエージェントを一定範囲で試せます。ただし利用回数や高度な機能には上限があり、Pro等の有料プランで拡張できます。最新の枠と料金は公式で確認してください。

WindsurfとCursorの違いは何ですか？

どちらもVS Code系のAI統合エディタですが、開発元が異なります。WindsurfはCodeium製でCascadeエージェントと無料枠の充実が強みです。CursorはAnysphere製でエージェント機能や有料プランの構成が異なります。試験では開発元と製品名をセットで覚えます。

WindsurfとCodeium拡張機能は同じですか？

いいえ。Codeium拡張機能はVS CodeやJetBrains等の既存エディタに追加する補完プラグインです。WindsurfはCodeiumが提供する独立したAIエディタ本体で、Cascadeエージェントなどエディタ全体に統合された機能があります。名称が似ているため混同しやすい点に注意してください。

Windsurfは何社が提供していますか？

Codeium（コーディアム）が提供するAIコードエディタです。同社はVS Code向けのCodeium拡張機能でも知られますが、Windsurfは拡張機能ではなく単体のエディタ製品として位置づけます。

WindsurfとGitHub Copilotの違いは？

GitHub Copilotは主に既存エディタ内でのコード補完・チャット支援です。Windsurfはエディタ本体にAI（Cascade含む）を組み込んだ製品です。Microsoft CopilotやGitHub Copilotとは名称・提供元・用途が異なるため、試験では混同しないよう整理します。
