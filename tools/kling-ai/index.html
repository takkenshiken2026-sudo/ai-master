試験で問われる見方

生成AIパスポートでは動画生成AIのカテゴリと利用上の注意が問われます。Kling AIはRunwayやVeoと並ぶ「動画生成AIの具体例」として、クレジット制・商用条件の確認が試験の定番論点です。

「動画生成AI = すべて同じ料金体系」という誤解に注意してください。Klingはクレジット消費がモデル・尺・品質モードで変わるため、ツールごとに制度を区別して覚える必要があります。

Kling AIとは

Kling AI（可灵）は、短视频プラットフォーム快手（Kuaishou）が展開する生成AIサービスです。Webブラウザとモバイルアプリから、テキストや画像を入力して動画を生成できます。

モデルはKling 1.xから2.x、2.6など世代が進んでおり、Professionalモードやネイティブ音声付き生成など高品質オプションが追加されています。開発者向けにはAPIも提供され、秒・解像度・モデルごとにクレジット（ポイント）が課金されます。

試験では「中国発の動画生成AI」「Runway・Pikaと並ぶ選択肢」として名前が出ることがあり、開発元と課金の仕組みをセットで押さえると得点しやすくなります。

できること（主な機能）

Text-to-Video

テキストプロンプトから5〜10秒程度の動画を生成。カメラワークや被写体の動きを指示できます。

Image-to-Video

1枚の画像や開始・終了フレームから動画化し、静止画素材を動きのあるクリップに変換できます。

画像生成・編集

Text-to-Imageや画像編集機能も統合され、動画制作の前段階を同一サービスで進められます。

リップシンク・音声

アバターやリップシンク、テキスト読み上げなど、音声まわりの機能モデルも提供されています。

Professionalモード

高品質生成モードはクレジット消費が増えますが、本番に近い仕上がりを狙えます。

Developer API

公式APIから動画生成を組み込め、モデル・尺・解像度ごとのクレジット単価が公開されています。

よくある誤解

代表的な誤解は「Kling AI = 月額定額で無制限」です。Klingはクレジット制であり、生成1回ごとにモデル・尺・品質に応じて消費します。会員プランは月次クレジットの付与枠が異なるだけで、使い切れば追加購入や上位プランが必要になります。

もう1つは「動画生成AIはすべて欧米製」という見方です。Kling AIは中国企業が提供する主要な動画生成サービスの一つです。試験では地域や開発元の違いより、クレジット制・利用規約・商用可否の確認が問われやすい点に注目してください。

料金・クレジット（2026年6月時点）

Kling AIはクレジット（Credits）を消費する方式です。無料枠、会員サブスクリプション、クレジット購入の3層で理解すると整理しやすくなります。以下は公式ドキュメントに基づく2026年6月時点の整理です。

クレジットの基本ルール（公式）

取得方法 有効期限（公式） 備考
会員サブスクリプションの月次クレジット配布から1か月プラン名・付与数は支払い画面に表示
クレジット購入配布から2年間標準レート：$1 USD = 66 Credits
ログイン・プロモーションクレジットイベントごとに異なる付与数は公式告知に従う

会員プラン（Standard / Pro / Premier / Ultra など）の月額・付与クレジット数は地域やキャンペーンで変動するため、契約前に klingai.com の会員ページで必ず確認してください。公式は「実際の金額は支払い画面の表示が優先」と明記しています。

動画生成の消費イメージ

1本あたりの消費クレジットは、モデル世代・尺（5秒/10秒）・Standard/Professional・音声の有無で変わります。Developer APIの公式Pricing（2026年6月時点）では、例としてKling 2.6のProfessionalモード・5秒・音声なしで1.5クレジット、10秒・音声付きで5〜10クレジットといった区分が公開されています。

Webアプリの表示とAPIの単価表は一致しない場合があるため、実際の利用前にアプリ内の消費表示を確認してください。

はじめ方・基本的な使い方

最短で試す流れは次のとおりです。

  1. アカウント登録 klingai.comでアカウントを作成し、利用可能なクレジット残高を確認します。
  2. 生成タイプを選択 Text-to-VideoかImage-to-Videoかを決め、尺とアスペクト比を設定します。
  3. プロンプトを入力 被写体・動き・カメラ・スタイルを具体的に記述します。
  4. モードを選ぶ 試作はStandard、本番候補はProfessionalなど、クレジットと品質のバランスを取ります。
  5. 商用前に規約確認 ウォーターマークの有無、商用可否、クレジット有効期限を公式ドキュメントで確認します。

よくある質問

Kling AIは無料で使えますか？

無料枠があります。ログイン時のデイリークレジットなどで試せますが、付与数やウォーターマークの有無はプランやキャンペーンで変わります。継続利用は有料会員またはクレジット購入が必要になることが多いです。

Kling AIのクレジットはどう買えますか？

公式Credits Policyによると、標準レートは $1 USD = 66 Credits です。購入クレジットは配布から2年間有効です。会員の月次クレジットとは別枠です。

Kling AIとRunwayの違いは何ですか？

どちらも動画生成AIですが、開発元・モデル・料金体系・編集機能が異なります。用途と課金モデルの両方で比較してください。

Kling AIは商用利用できますか？

商用可否はプランと利用規約に依存します。案件利用前に最新のTerms of ServiceとCredits Policyを必ず確認してください。

1本の動画に何クレジットかかりますか？

モデル・尺・品質モード・音声の有無で変わります。生成前にアプリ内表示を確認し、API利用時はDeveloper Pricingを参照してください。
