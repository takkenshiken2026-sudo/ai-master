Microsoft 365 Copilotとは？機能・料金・使い方をわかりやすく解説

Word・Excel・Teamsの「今開いている画面」で、社内メールと議事録を踏まえて動く法人向けAI

Graph接地 Work IQ 法人アドオン
対応環境
  • Word · Excel · PowerPoint
  • Outlook · Teams · OneNote
  • m365.cloud.microsoft
更新日：2026年6月10日 読了目安：約9分

Microsoft 365 Copilot（エムスリーオフィス・コパイロット）は、Microsoftが提供する法人向けAIアシスタントです。2023年11月の一般提供以降、Word・Excel・PowerPoint・Outlook・Teamsなどすでに使っているアプリの中で、メール・ファイル・会議・チャットを踏まえた回答や下書きを生成します。本記事は機能一覧の暗記ではなく、Microsoft Copilot（無料チャット）・Copilot Pro（個人）・M365 Copilot（法人）の三層ライセンス、Work IQとMicrosoft Graphによる業務データ接地、2026年6月のM365パッケージ更新（Copilot Chat enhancements）に絞って書いています。料金・同梱範囲は2026年6月時点の公式情報に基づき、契約前に必ず再確認してください。

試験で問われる見方

生成AIパスポートでは、Microsoft製AIはTF-0143（主要生成AI支援）の文脈で登場します。試験で効くのは次の3点です。

  • 名称の区別 — 「Microsoft Copilot」（無料Webチャット）と「Microsoft 365 Copilot」（法人アドオン）、さらに「GitHub Copilot」（コード補完）を混同しない
  • 業務データの扱い — 個人の無料アカウントに社内資料を入れない。法人利用は組織契約とガバナンス下で行う（TF-0289・TF-0350）
  • 接地の仕組み — M365 CopilotはMicrosoft Graph経由でメール・ファイル等を参照する「業務データ接地型AI」の代表例

Microsoft 365 Copilotとは

Microsoft 365 Copilotは、チャットボットを別タブで開くのではなく、Wordのリボン、Excelのセル横、Teamsの会議画面など、作業中のUIにCopilotアイコンとして埋め込まれたAIです。2023年11月にEnterprise向けGA、2024年以降Business向けも拡大。2026年時点ではWork IQという知能レイヤー、Copilot Search、Copilot Notebooks、Copilot Create、Copilot Studioによるエージェントなど、単なる「要約ボタン」を超えた機能群に発展しています（Microsoft公式）。

技術的には、ユーザーの質問に対しMicrosoft Graph上のメール・Teamsチャット・SharePoint/OneDriveファイル・カレンダー等を検索し、取得した文脈をLLM（2026年3月時点でGPT-5.4等が利用可能と発表）に渡して回答を生成するRAG（検索拡張生成）型のパイプラインです。重要なのは、ユーザーがもともと閲覧権限を持つデータだけが参照対象になる点——これが個人向けWeb Copilotとの決定的な差です。

三層ライセンスの整理

MicrosoftのCopilotブランドは名称が似ており、試験でも実務でも混乱しやすいです。本記事のオリジナリティは、この三層表にあります。

層 製品名 主な利用者 業務データ接地 アプリ内Copilot
第1層 Copilot Chat（無料Web等） 個人・一部M365ユーザー 限定的（2026年6月以降M365にenhancements同梱） ×（Webチャット中心）
第2層 Copilot Pro 個人（約$20/月） 個人Microsoft 365データ（条件付き） Office Web版の一部（条件付き）
第3層 Microsoft 365 Copilot 法人（Business/Enterprise） Graphフル（テナント全体） Word/Excel/Teams等フル

「家で試した無料Copilot」と「会社のM365 Copilot」は参照できるデータ・管理者の制御・課金単位が異なります。業務判断ではIT部門の案内を優先し、Microsoft Copilot記事（第1〜2層）と本記事（第3層）を分けて読むと整理しやすいです。

Work IQとMicrosoft Graph

2026年時点のMicrosoft公式は、M365 Copilotの背後にWork IQという知能レイヤーがあると説明しています。Work IQは大きく3要素で構成されます。

Data（データ）

メール、ファイル、会議、チャット、業務トランザクション——組織内の「仕事の実態」をGraph上から取得。

Context（文脈）

ユーザーの文体・嗜好・習慣を記憶し、時間とともにパーソナライズ（Microsoft公式説明）。

Skills & tools

データと文脈を組み合わせ、特定タスク向けにCopilotとエージェントを最適化。

権限継承とデータプライバシー

Microsoft公式は次を明記しています（2026年6月時点）。

  • プロンプト・入力・応答はモデル学習に使用しない
  • CopilotはM365の権限・機密ラベル・保持ポリティを継承——見えないファイルは参照されない
  • 管理者向けITツールでCopilotとエージェントをテナント単位で管理可能

試験では「AIだから全社データを見られる」という誤解を避けることが重要です。Graph接地は権限の範囲内での参照であり、情報セキュリティ設計とセットで理解します。

アプリ内でできること

M365 Copilotライセンスが必要な機能の代表例です（プラン・ロールアウト状況は変更あり）。

Word

既存文書や指定ファイルを踏まえた下書き・要約・トーン変更。Agent Modeで複数ステップ編集（2026年ロールアウト）。

Excel

自然言語での数式提案、データ分析、グラフ生成、Python in Excel連携（環境依存）。

PowerPoint

Word文書やプロンプトからスライド生成、デザイン調整、Speaker notes作成。

Outlook

メールスレッド要約、返信下書き、トーン調整。受信箱・カレンダー文脈を踏まえた提案。

Teams

会議要約、リアルタイム字幕、チャット要約、会議中の質問応答（Intelligent recap等）。

Copilot Search / Notebooks / Create

自然言語の社内検索、プロジェクト素材の統合分析、資料・動画・ポッドキャスト風要約の生成。

Copilot Studioとエージェント

M365 CopilotライセンスにはCopilot Studioへのアクセスが含まれ、低コードで業務エージェント（契約レビュー、CRM参照、オンボーディング等）を構築できます。Agent StoreからMicrosoft/partner製のReady-to-use agentsも利用可能です。エージェントはWork IQとGraphコネクタで外部データソースにも接続できますが、IT設定とメータリングが必要な場合があります。

2026年のパッケージ更新

2025年12月4日、MicrosoftはM365商用スイートの価格・パッケージ更新を発表しました（Microsoft Licensing Resources）。2026年6月以降、テナントごとに30日通知のうえ段階的にロールアウトされ、2026年8月1日までに完了予定です。

本記事で押さえるべきは、Copilot Chat enhancementsがコアM365スイート（E3/E5/Business等）に同梱される点です。enhancementsには受信箱・カレンダー認識とWord/Excel/PowerPointエージェントへのアクセスが含まれます（Microsoft公式注釈）。

ただしこれはフルM365 Copilotアドオンと同一ではありません。Teams会議インテリジェンスのフル機能、Copilot Search、Copilot Studio、管理ダッシュボード等は引き続きMicrosoft 365 Copilotライセンスが必要です。導入検討時は「スイート同梱分で足りるか」「アドオンが必要か」をITと切り分けて確認してください。

なお、Standalone Copilot SKU（M365 Copilotアドオン単体）は今回のリスト値上げ対象外と明記されています（2026年7月1日のM365本体値上げとは別枠）。

よくある誤解

「M365を契約していればCopilotが使える」——基本ライセンスだけではアプリ内Copilotは使えません。Copilot Chat enhancementsの同梱範囲とフルアドオンは別物です。

「Copilot＝GitHub Copilot」——GitHub CopilotはIDE向けコード補完。名称は似ていますが製品・課金・用途が異なります。

「Graph接地ならSlackやGoogle Driveも見える」——標準はM365データです。Slack、Jira、Salesforce等はGraphコネクタの追加設定が必要で、未対応のオンプレ製品もあります。

「全員にライセンスを割り当てるべき」——$21〜$30/ユーザー/月にM365本体が上乗せされるため、全社一斉より高頻度利用者から段階展開が現実的です。Microsoftも利用分析ダッシュボードを提供しています。

導入の流れ・使い方

法人利用は個人のWeb Copilot試用とは手順が異なります。

  1. 前提ライセンスの確認 Business Standard/Premium、E3/E5等、Copilot追加の資格を満たすか確認します。
  2. 管理者がCopilotライセンスを購入・割当 Microsoft 365管理センターからユーザーまたはグループにライセンスを割り当てます。
  3. データ境界・Graph設定の確認 機密ラベル、DLP、保持ポリシー、Copilotのデータ処理オプションをITがレビューします。
  4. アプリ内Copilotアイコンから利用 Word/Excel等でCopilotペインを開き、「/」コマンドや自然言語で依頼します。
  5. 出力は人が確認してから共有 数式・引用・法務文書はハルシネーションリスクがあるため、必ず人間が検証します（TF-0350）。

うまく使うためのコツ

  • 参照ファイルを明示する — 「/」や@メンションで特定の社内文書を指定すると精度が上がりやすい
  • Teams会議の議事録を起点にする — 会議要約→Word下書き→Outlook返信の流れがM365 Copilotの定番
  • エージェントは小さく始める — Copilot Studioで1業務（例：FAQ応答）からPilot展開
  • 採用分析を見る — 使われていないライセンスは他ユーザーへ再配分を検討

  • すでにMicrosoft 365 E3/E5またはBusiness Standard以上を全社利用している組織

よくある質問

Microsoft 365 Copilotと無料のCopilot Chatの違いは？

Copilot ChatはWebチャット中心で、2026年6月以降M365スイートにCopilot Chat enhancements（受信箱・カレンダー認識等）が段階的に同梱されます。Microsoft 365 CopilotはWord・Excel・Teams等アプリ内AIとMicrosoft Graphによる業務データ接地、Copilot Search、Copilot Studio等を含む有料アドオンです。

Microsoft 365 Copilotの料金はいくらですか？

2025年12月以降、Business向けは$21/ユーザー/月（年払い）、Enterprise向けは$30/ユーザー/月（年払い）が目安です。Microsoft 365 Business Standard/PremiumやE3/E5等の基本ライセンスが別途必要で、合算すると$33〜$87/ユーザー/月程度になります（2026年6月時点）。

Microsoft 365 CopilotとCopilot Proの違いは？

Copilot Proは個人向けの上位プラン（約$20/月）で、Office Web版の一部AI等が対象です。Microsoft 365 Copilotは法人向けアドオンで、テナント全体のGraph接地・権限継承・管理ポータル・Teams会議インテリジェンス等が含まれます。

Microsoft 365 Copilotは社内データを学習に使いますか？

Microsoft公式は、プロンプト・入力・応答をモデル学習に使用しないと明記しています。さらにCopilotはMicrosoft 365の権限・機密ラベル・保持ポリシーを継承するため、ユーザーが閲覧できないファイルは参照されません（2026年6月時点）。

Microsoft 365 CopilotとNotion AIの違いは？

Notion AIはNotionワークスペース内のページ・DB・連携アプリが中心です。Microsoft 365 CopilotはExchangeメール、SharePoint、Teamsチャット、Outlookカレンダー等Microsoft Graph上の業務データを前提に、Word/Excel/Teams等の既存アプリ内で動作します。組織の基盤がMicrosoftかNotionかで選び方が分かれます。
