試験で問われる見方

生成AIパスポートでは、RAG、一次情報の確認、AI生成物の検証が問われます。Consensusは「論文データベースを参照して回答する」学術RAGの例として理解すると、情報リテラシーの具体例になります。

「Meterの結果＝医学的結論」「Consensus＝Google Scholar」と混同しないことが重要です。エビデンスの質と研究デザインの評価は人間の仕事です。

Consensusとは

Consensusは、AI-powered academic search engineを標榜するWebサービスです。2億本以上（公式では最大2.5億本規模とも案内）の査読付き論文を検索し、研究質問に対するエビデンスを要約・可視化します（公式サイト）。

Consensus Meterは、はい/いいえ形式の研究質問に対し、論文がどれだけ賛成・反対・中立かをグラフで示します。Deep Searchは検索戦略の拡張・引用グラフ探索など、文献レビューを自動化するモードです（公式サイト）。

試験では、ConsensusをChatGPTのような汎用AIではなく、論文エビデンス特化の検索アプリとして区別して覚えるとよいです。

できること（主な機能）

Papers Search

論文データベースの意味検索。Freeでも月間無制限（公式ヘルプ）。

Consensus Meter

Yes/No研究質問へのエビデンス賛否を可視化。素早いエビデンス確認に向く。

Deep Search / Deep Review

包括的な文献レビュー戦略を自動構築。Freeは月3回（公式ヘルプ）。

自然言語フィルタ

「RCTのみ」「過去5年」などをプロンプトで指定し自動適用（公式サイト）。

Medical mode

臨床ガイドライン・主要医学誌に絞った検索モード（公式サイト）。

Study Snapshots

論文の要点をスナップショット化。Freeは月10回（公式ヘルプ）。

よくある誤解

代表的な誤解は「Consensus Meter＝最終的な医学的結論」です。Meterは論文の傾向を可視化するものであり、個別研究の質やバイアス評価は別途必要です。

もう1つは「Consensus＝Elicit」です。どちらも論文AIですが、Consensusは賛否の可視化・Deep Search、Elicitはシステマティックレビュー・大規模抽出が強みです。

「論文検索＝必ず正しい回答」とも限りません。出版バイアスや研究の限界は常に存在します。TF-0388のように、AI出力は人が検証します。

はじめ方・基本的な使い方

  1. アカウント作成consensus.appで無料登録します。
  2. 質問を入力「Ask the research...」に研究質問を自然文で入力します。
  3. モード選択Deepをオンにすると深い文献レビュー、Medical modeで臨床向けに絞れます。
  4. Meterを確認Yes/No質問では賛否グラフを確認し、根拠論文を開きます。
  5. 引用を検証重要な結論は原論文で確認します。

よくある質問

無料で使えますか？

はい。論文検索は無制限です。AI分析機能には月次上限があります。

Consensus Meterとは？

Yes/No質問への論文エビデンスの賛否を可視化する機能です。

Elicitとの違いは？

Consensusは賛否可視化、Elicitは大規模文献レビューが強みです。

ChatGPTの代替になりますか？

論文調査には向きますが、汎用チャットはChatGPT等と使い分けが必要です。

Proはいくらですか？

$15/月、または年払い$120/年（月$10相当）です。
