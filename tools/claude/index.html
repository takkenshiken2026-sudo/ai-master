試験で問われる見方

生成AIパスポートでは、Claudeは「Anthropicが提供する主要な生成AIサービス・モデル群」として扱われます。ChatGPT（OpenAI）やGemini（Google）と並ぶ代表例として、名称と開発元の組み合わせがそのまま出題されます。G検定でも生成AIの文脈で主要サービスとして触れることがありますが、深掘りの仕方は試験ごとに異なります。

よくある誤解は2つあります。1つ目は「Claude＝CNNの畳み込み」のような技術用語と混同すること（Claudeはサービス名です）。2つ目は「Claude＝AI全体」と捉えること。試験では「どの会社の、どんな種類のツールか」を短く説明できるかが問われます。実務でも、ツール名だけ覚えて開発元やデータ取り扱いを確認しないと、契約・ガバナンスの判断を誤りやすくなります。

Claudeとは

Claudeは、2023年から順次公開されたAnthropic製のAIチャットサービスです。人名の「クロード・シャノン」にちなんだ名称で、背後では大規模言語モデル（LLM）が動いています。モデル名は時期により更新されるため、試験勉強では「最新の型番」より開発元・用途・リスクを優先して覚えると安定します。

Anthropicは「安全性を重視したAI開発」で知られ、回答が拒否されたり、注意書きが付いたりする場面がChatGPTより目立つことがあります。これは不具合ではなく、有害・不正利用を抑える設計の表れです。生成AIパスポートのケース問題では、「なぜ業務利用に社内ルールが必要か」とセットで考えると理解が深まります。

機能面の印象として強いのは長い資料を一度に読ませて要約・分析できることと、ビジネス文書らしい丁寧な文体です。画像生成や音声会話の充実度はChatGPTに譲る場面もありますが、「読む・書く・考える」作業の相棒として選ばれることが多いツールです。

Claudeでできること（主な機能）

Claudeも汎用チャットAIですが、得意な作業には傾きがあります。資格勉強では「何ができるツールか」をカテゴリで整理しておくと比較問題に強くなります。

長文の要約・分析

PDFや長いレポートをアップロードし、要点抽出・論点整理・リスク洗い出しを依頼できます（プランにより上限あり）。

文章の起草・推敲

メール、報告書、提案書の下書きや、トーン調整・敬語の統一など、読み手を意識した文章生成に強みがあります。

コードの生成・説明

Python、SQL、JavaScriptなどのコード作成や、既存コードの解説・リファクタリング案を出せます。

Projects（プロジェクト）

資料や指示をまとめて保持し、同じ文脈で繰り返し作業する機能（有料プラン中心）。業務の「案件単位」で使いやすいです。

画像の理解

チャートやスライド画像の内容説明など、画像入力による分析に対応。ただし画像生成の主役は他ツールのことが多いです。

Web検索連携

最新情報が必要な質問で、検索結果を踏まえた回答を得られる場合があります。回答は依然として要確認です。

コンテキストウィンドウ（長文対応）

Claudeの差別化ポイントのひとつが、一度に扱えるテキスト量の大きさです。契約書や研究資料のようにページ数が多い文書を、分割せずに渡せる場面があります（上限はモデル・プラン依存）。試験では数値暗記より「長文に強いチャットAIの代表例」として覚える程度で十分なことが多いです。

はじめ方・基本的な使い方

Claudeも、ブラウザまたはアプリから数分で始められます。

  1. アカウントを作成する claude.aiにアクセスし、メールまたはGoogleアカウントで登録します。
  2. 目的を一文で伝える 例：「以下の議事録から決定事項とToDoだけ箇条書きで抽出して」
  3. 資料がある場合は添付する PDFやテキストをアップロードし、分析範囲（章・ページ）を指定すると精度が上がります。
  4. 回答を確認し、追質問する 「根拠はどの段落か」「反対意見も述べて」など、会話を続けて深掘りできます。

うまく使うためのコツ

  • 読者と目的を書く — 「経営層向けに1ページ要約」「法務チェック用の論点リスト」など
  • 資料のどこを見るか指定する — 長文ほど「第3章のみ」「表だけ」など範囲を絞る
  • 出力形式を決める — 表形式、箇条書き、メール文などフォーマットを先に指定
  • 事実は必ず人が確認する — 引用・数値・法務判断はAI任せにしない

ビジネスでの活用例

Claudeが特に選ばれやすいのは、「読む量が多い」「文章の質が重要」な業務です。

法務・コンプライアンス

  • 契約書ドラフトの論点整理（最終判断は人間）
  • 社内規程との整合チェックの下準備
  • ポリシー文書の平易化

企画・コンサル

  • 長い調査レポートの要約
  • 提案書の構成案・Executive Summary
  • 競合情報の整理（公開情報ベース）

エンジニア・データ

  • 仕様書からの要件抽出
  • コードレビュー補助
  • 技術文書の翻訳・整形

人事・総務

  • 就業規則・マニュアルの要約
  • 社内アンケート結果の傾向整理
  • 研修資料のたたき台作成

いずれも個人情報・未公開情報の入力は社内ルールに従うことが前提です。生成AIパスポートでは、こうしたガバナンスの判断がケース形式で問われます。

よくある質問

Claudeは無料で使えますか？

はい。無料プランで基本的なチャット機能を利用できます。ただし、1日の利用量や使えるモデルに制限があり、長文の大量処理や高度な機能は有料プラン（Pro以上）向けです。

Claude Proはいくらですか？

Claude Proは月額20ドル前後（為替で約3,000円前後）が目安です。高性能モデルへの優先利用や、より多くのメッセージ送信が可能になります。最新料金は公式サイトでご確認ください。

ClaudeとChatGPTの違いは何ですか？

ClaudeはAnthropic製で、長文の読み込みや丁寧な文章生成・分析に強みがあります。ChatGPTはOpenAI製で、画像生成・音声会話・カスタムGPTなど機能の幅が広いのが特徴です。用途に応じて使い分けるのがおすすめです。

Claudeは何社が提供していますか？

Anthropic（アンソロピック）が提供する主要な生成AIサービス・モデル群です。試験では開発元とサービス名をセットで覚えることが多いです。

会社の業務でClaudeを使っても大丈夫ですか？

個人の無料アカウントに機密情報を入力するのは避けるべきです。業務利用ではTeam・Enterpriseプランの検討と社内ガイドラインの整備が推奨されます。データの学習利用ポリシーはプランごとに異なるため、利用前に公式の規約を確認してください。
