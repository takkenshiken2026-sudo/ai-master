Haiper AIとは？動画生成AIの機能・料金・使い方を解説

秒単位クレジットで尺を見積もるText-to-Video——クリエイター向けの短尺動画生成

Text-to-Video Haiper 2.0 Web · iOS · API
対応環境
  • Web（haiper.ai）
  • iOSアプリ
  • API
更新日：2026年6月10日 読了目安：約8分

Haiper AI（ハイパー）は、Haiper社が提供する生成AI動画プラットフォームです。Google DeepMind出身の研究者が2020年代に立ち上げ、Text-to-Video（テキストから動画）とImage-to-Video（画像から動画）をブラウザとiOSで試せるサービスとして注目を集めました。2024年10月のHaiper 2.0はリアリズムと生成速度を前面に出し、Video TemplatesでSNS・広告向けの型も提供しました。本記事はモデル一覧の暗記ではなく、「秒単位クレジット」という課金設計、Runway・Pika・Veo 3との棲み分け、試験で問われる動画生成AIの位置づけに絞って書いています。2026年6月時点では consumer 向けWebの提供状況が変化している可能性があるため、利用前はサービス状況と公式ドキュメントを必ず確認してください。

試験で問われる見方

生成AIパスポートでは、動画生成AIは「テキストや画像から短尺動画を作る技術」として説明されます（TF-0219）。HaiperはText-to-Videoの具体サービスの一例として整理すると、拡散モデルやマルチモーダルAIの章とつながります。

よくある誤答パターンは、「動画生成AI＝文章要約・翻訳が得意なチャットAI」（×）です（HQ-0421）。Haiperは映像出力が主目的であり、LLMチャットそのものではありません。また「生成できれば商用・真正性の確認不要」も×——TF-0348のように利用規約・権利・ディープフェイクリスクの確認が試験でも実務でも必要です。

Haiper AIとは

Haiperは2021年頃にロンドンで創業した、知覚（perception）基盤モデルと動画生成を研究・製品化するスタートアップです。共同創業者のDr. Yishu Miao（CEO）とDr. Ziyu Wang（CTO）は、Oxford大学で機械学習のPhDを取得し、Google DeepMindで研究経験を持つと公式プレスリリースで紹介されています（2024年10月）。

2024年3月に一般公開された当初から、ブラウザだけでプロンプト入力→短尺MP4出力という手軽さを訴求しました。公開後7か月で450万ユーザーを超えたと同年10月の発表があり、クリエイター層への浸透の速さが話題になりました。技術面では、公式はTransformerと拡散（diffusion）を組み合わせた独自モデルで、Haiper 2.0はリアリズム・時間的一貫性・生成速度を改善したと説明しています。

試験では、Haiperを「動画生成AIのサービス名」と「Text-to-Video / Image-to-Videoの実装例」として覚えると、RunwayやSoraとの比較問題で混乱しにくくなります。

秒単位クレジットの考え方

Haiper理解で他ツールと最も異なるのが、動画生成のクレジットが「秒」に比例する点です。公式ドキュメント（docs.haiper.ai/pricing）では、Haiper Video 2.xの720p Text-to-Videoが8クレジット/秒、540pが5クレジット/秒と表記されています。

課金単位 Haiper（例：Video 2.x 720p） Runway / Pika（一般的傾向）
基本単位 生成秒数 × 解像度別レート 1生成あたり固定クレジット（機能・モデル依存）
見積もり 8秒クリップなら 8×8＝64cr（720p） 「1本いくら」で概算しやすい
試験・実務 尺を延ばすほどコスト増 高機能1回が高額になりうる

8秒のSNSクリップを720pで量産する場合、秒課金は「長尺ほど痛い」と直感できます。逆に2〜4秒のテスト生成なら、1本あたりのクレジットは比較的抑えられます。Extend（尺延長）やEnhance（1080p化）を組み合わせると、表の単価に加えて追加操作分が載るため、本番前にdocsのCredits表で総コストを試算してください。

静止画（Haiper Image）

Haiperは動画だけでなくText-to-Imageも同一アカウントで扱います。Image 2.xは720pで5クレジット/4枚、4Kで20クレジット/4枚（公式Pricing）。動画のキービジュアルをImageで作り、Image-to-Videoで動かす——という同一プラットフォーム内パイプラインがHaiperの特徴の一つです。

できること（主な機能）

Text-to-Video

プロンプトから短尺MP4を生成。16:9・9:16・1:1などSNS比率に対応（プラン依存）。

Image-to-Video

静止画にカメラワークや被写体の動きを付与。商品写真・イラストの動画化向き。

Video Repaint

既存クリップのスタイル・色調を変更。部分リライトでバリエーション探索。

KeyFrame Conditioning

開始・終了フレームを指定し、中間の動きを補間。ストーリーボードに近い指示が可能。

Extend / Enhance

尺延長（Extend）と1080p化（Enhance）。Video 1.5系で提供（公式Pricing）。

Video Templates

Haiper 2.0と同時公開。トレンド動画・商品広告の型を選んで生成（公式PR）。

API（アプリ課金とは別）

開発者向けAPIは、Web/iOSのサブスククレジットとは別の従量課金です。公式API Pricingでは、Video 2.x 720p Text-to-Videoが$0.05/秒、540pが$0.033/秒（2026年6月時点・docs.haiper.ai）。自社プロダクトに組み込む場合は、consumerプランとAPIキー課金を混同しないことが重要です。

2026年6月時点のサービス状況

試験対策としてHaiperの技術的位置づけを学ぶ一方、実際に触れるかは別問題です。2026年6月10日時点で当サイトが確認した範囲では、haiper.aiのトップページはロゴ表示のみで生成UIに到達できず、/membershipは404でした。一方、docs.haiper.aiにはPricing・APIリファレンスが残存しています。

報道ベースでは、2025年以降に創業者がMicrosoft等へ移り、consumer向けサービスの縮小・モデル技術の譲渡が言及される記事もあります（第三者メディア）。試験では「こういう動画生成サービスがあった／ある」として押さえ、本番利用前は必ず公式の稼働状況を確認してください。代替として現役のRunway、Pika、Kling AIでText-to-Videoの実習を続けるのが安全です。

よくある誤解

「Haiper＝Runwayの完全下位互換」——どちらもText-to-Videoですが、RunwayはGen-4系と編集タイムライン統合でプロ制作フローに寄り、Haiperは短尺・秒課金・テンプレートでクリエイター試作に寄せた設計でした。画質最優先ならRunwayやVeo 3、手軽な尺試しならHaiper——と用途で分けます。

「無料＝ウォーターマークなし・商用OK」——公式Termsは、Free tierに制限があり、有料購読で追加機能と制限解除（ウォーターマーク・商用・非公開等）が行われる設計です。Proプランでウォーターマークなし・商用利用が明記されていた時期もありますが、プラン名・条件は変動するため、案件前に必ず最新Termsを読んでください。

「動画生成AI＝Deepfakeツール」——技術的に顔差し替え等と近い拡散系も使われますが、試験では用途と倫理・権利がセットです。Haiperで生成した人物映像をそのままニュース風に使う等は、TF-0220が示すリスク確認の典型例になります。

はじめ方・基本的な使い方

サービスが利用可能な場合の、クレジット効率と品質を両立する手順です。

  1. 稼働確認 haiper.ai または iOSアプリにログインできるか確認。UIが利用できない場合はAPIドキュメントまたは代替ツールを検討します。
  2. 尺と解像度を決める 720p・8秒なら Video 2.x で 64クレジット（8×8）が目安。まず540pや短尺で試します。
  3. プロンプトまたは画像を入力 被写体・カメラ・照明・動きを具体化。Image-to-Videoなら構図が決まった静止画を起点にします。
  4. Templates / Repaintで調整 SNS向けテンプレートやRepaintでスタイルを揃え、Extendで尺を足すか検討します。
  5. 商用前にプランとTermsを確認 クライアント納品・広告配信はPro以上とライセンス条項を必ず確認（TF-0348）。

よくある質問

Haiper AIは無料で使えますか？

過去には無料枠（日次生成上限と非失効クレジット）でText-to-Videoを試せました。2026年6月時点ではhaiper.aiのWeb UIが制限されている可能性があるため、利用前に公式サイトとdocs.haiper.aiで最新状態を確認してください。

Haiper AIの強みは何ですか？

Text-to-Video・Image-to-Videoをブラウザ/iOSで手軽に試せた点、秒単位のクレジット課金で尺を見積もりやすい点、Haiper 2.0以降のリアリズムと生成速度、動画と静止画を同一プラットフォームで扱える点が特徴として挙げられます。

Haiper AIとRunwayの違いは？

どちらもText-to-Videoですが、RunwayはGen-4系と編集タイムライン統合が強くプロ向け、Haiperは短尺クリップの素早い試作と秒単位クレジット、クリエイター向けテンプレート訴求が強い傾向でした。商用・品質要件が高いならRunway、手軽なSNS尺の探索ならHaiper——という整理が参考になります。

Haiper AIはText-to-Videoですか？

はい。テキストプロンプトから短尺動画を生成するText-to-Videoが中核です。Image-to-Video、Video Repaint、KeyFrame Conditioning、Extendなども提供され、試験の「動画生成AI」（TF-0219/TF-0220）の具体例になります。

Haiper AIは商用利用できますか？

商用利用・ウォーターマークなし・非公開生成は有料プラン（Pro等）で解放される設計でした。Free枠は制限付きです。案件利用前にTerms of Useとプラン表を必ず確認してください（TF-0348）。
