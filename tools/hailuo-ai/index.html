Hailuo AI（ハイルオ / 中国語名：海螺视频）は、中国のAI企業MiniMaxが提供する動画生成AIサービスです。テキストや画像から6〜10秒の動画を生成し、Web上では贝壳（シェル）と呼ばれるクレジットを消費します。本記事では、MiniMax三層エコシステム、贝壳＋低速無制限という独自の課金設計、Hailuo 2.3のモデル階梯、Kling・Runwayとの比較を整理します。料金と機能は2026年6月時点の情報です。

Hailuo AIとは

Hailuo AI（hailuoai.com/video）は、MiniMaxが展開する動画生成サービスです。中国語では「海螺视频（ハイルオ・シーパン）」と呼ばれ、テキストプロンプトや画像から短尺動画を生成します。

2025〜2026年にかけてHailuo 02からHailuo 2.3 / 2.3-Fastへとモデルが更新され、身体動作・表情・物理リアリティ・プロンプト追従の改善が公式に訴求されています。解像度は768P・1080P、尺は6秒・10秒が中心です。

試験では「MiniMax発の動画生成AI」「Kling・Pika・Runwayと比較される中国系ツールの一つ」として名前が出ることがあります。開発元と課金単位（贝壳）を区別して覚えることが重要です。

MiniMax三層エコシステム

MiniMaxブランドで混同されやすいのが、一般向けHailuoと開発者向けOpen Platformの二系統です。試験・実務とも、同じ社名でも製品ラインを分けて理解してください。

レイヤー 入口 主な用途 課金の考え方
Hailuo AI（一般向け） hailuoai.com Web UIでT2V・I2V・Agent等 会員プラン＋贝壳（シェル）月次付与
MiniMax Open Platform API platform.minimax.io 動画・LLM・TTS・音楽・画像のAPI統合 Pay-as-you-go（動画は1本あたり$0.19〜）
同一ブランドの隣接製品 hailuoai.com（音声タブ等） Text-to-Speech・音声クローン 贝壳またはAPI従量（製品ごとに異なる）

動画生成だけを試すならHailuo AI、自社アプリに組み込むならOpen Platform API——この棲み分けが、Runway（Web＋API一体）やKling（Web中心）との比較軸になります。

MiniMax公式APIドキュメント（2026年6月時点）では、動画生成モデルが世代・用途別に整理されています。Webアプリでも同系統のモデル名が選択できます。

モデル 公式の位置づけ 解像度・尺 API従量（768P・6秒）
MiniMax-Hailuo-2.3 身体動作・表情・物理リアリティ・プロンプト追従の強化 768P / 1080P、6s / 10s $0.28
MiniMax-Hailuo-2.3-Fast Image-to-Video向け。効率・コスパ重視 768P / 1080P、6s / 10s $0.19
MiniMax-Hailuo-02 1080P・10秒・プロンプト追従（旧世代だが低解像度は安価） 512P〜1080P、6s / 10s $0.10（512P・6s）

Web会員プランでは、最新の2.3系は贝壳消費が前提で、旧世代（Hailuo 01/02）は上位プランで低速キュー無制限が付く設計があります（後述）。「最新＝高品質＝高コスト」「旧世代＝低速だが使い放題」という二層構造が、KlingやRunwayにはないHailuoの特徴です。

できること（主な機能）

Text-to-Video

テキストプロンプトから6〜10秒の動画を生成。被写体の動き・カメラワーク・スタイルを指示できます。

Image-to-Video

1枚の画像、開始・終了フレーム、参照画像から動画化。2.3-FastはI2V効率向けと公式に位置づけ。

1080P・10秒生成

Standard以上の有料プランで10秒1080P動画の生成・ウォーターマークなしDLが可能（公式プラン表）。

並列キュー

Standard以上で最大5タスク投入・2タスク同時並列。量産ワークフロー向け（公式プラン表）。

Agent・テンプレート

Agent機能やテンプレート動画生成も提供。贝壳消費対象（2026年6月時点）。

Developer API

非同期API（タスク作成→ステータス確認→ダウンロード）で自社サービスに組み込み可能。

よくある誤解

代表的な誤解は「Hailuo AI＝MiniMax API」です。一般向けWeb（贝壳＋会員）と開発者向けOpen Platform（従量課金）はアカウント・料金体系が異なります。APIの単価表をそのままWebプランに当てはめないでください。

もう1つは「上位プラン＝すべて無制限」です。尊享（Max）プランでも、無制限が適用されるのは主にHailuo 01/02など旧世代モデルの低速キューです。最新のHailuo 2.3系は贝壳消費が必要です（2026年6月時点の公式説明）。

3つ目は「中国発＝Kling AIと同じ」です。どちらも高品質な動画生成で知られますが、開発元・クレジット名称・プラン設計は別会社です。

贝壳クレジットと低速無制限

Hailuo AIのWeb課金の核は贝壳（バイケ / Shell）と呼ばれる月次クレジットです。1本あたりの消費量はモデル・解像度・尺で変わり、公式FAQでは例として「4500贝壳 ≒ 6秒768P動画を約180本（Hailuo 2.0モデル）」といった換算が示されています。

贝壳が尽きたとき（上位プラン）

尊享（Max）等の上位会員では、贝壳を使い切った後も低速キューでHailuo 01/02モデルの生成を継続できる場合があります。公式UIでは「贝壳被消耗光了→低速通道でHailuo01/02模型无限生成」と案内されるケースがあります（2026年6月時点）。

つまりHailuoの課金は「最新モデル＝贝壳消費の高速路」と「旧モデル＝低速だが使い続けられる保険」の二層構造です。量産テストや下書きには旧モデル、本番クオリティには2.3系——という使い分けが現実的です。

はじめ方・基本的な使い方

  1. アカウント登録hailuoai.comでアカウントを作成し、付与された贝壳残高を確認します。
  2. モデル選択Hailuo 2.3（高品質）か2.3-Fast（I2V効率）か、旧世代（02）かを用途で選びます。
  3. 入力Text-to-Videoならプロンプト、Image-to-Videoなら画像をアップロード。解像度・尺を設定。
  4. 生成・確認キューで生成完了を待ち、プレビューで動き・表情を確認。必要なら再生成。
  5. 書き出し有料プランでウォーターマークなしDL。商用案件は利用規約を再確認してから納品。

よくある質問

Hailuo AIとMiniMax APIの違いは？

Hailuo AI（hailuoai.com）は一般向けの動画生成Webサービスで、贝壳（シェル）クレジットと会員プランで利用します。MiniMax Open Platform API（platform.minimax.io）は開発者向けで、動画・LLM・TTS等を従量課金またはVideo Packagesで利用します。同じMiniMaxブランドでも製品ラインが異なります。

無料で使えますか？

無料トライアル用の贝壳（シェル）が付与される場合があります。継続利用はStandard以上の有料会員または贝壳の追加購入が必要です。無料枠の付与数・ウォーターマークの有無はキャンペーンで変わるため、登録時の表示を確認してください（2026年6月時点）。

Kling AIとの違いは？

どちらも中国発の動画生成AIですが、Hailuo AIはMiniMax（海螺视频）、Kling AIは快手（Kuaishou）が提供します。クレジット単位（贝壳 vs Credits）、モデル名、上位プランの「低速無制限」設計などが異なります。

贝壳がなくなるとどうなりますか？

上位会員プランでは、贝壳を使い切った後もHailuo 01/02など旧世代モデルで低速キューによる無制限生成が可能な場合があります（プラン・モデルにより異なる）。最新のHailuo 2.3系は贝壳消費が前提です（2026年6月時点）。

商用利用できますか？

商用利用の可否はプラン・利用規約・生成物の権利帰属に依存します。Standard以上でウォーターマークなし出力が可能なプランもありますが、案件利用前に公式Terms of Serviceを必ず確認してください。
