試験で問われる見方

生成AIパスポートやG検定では、動画生成AIの特徴と利用上の注意が問われます。Veo 3は「Google製の動画生成モデル」「音声を含む動画生成」という2点をセットで覚えると整理しやすいです。

試験では「Veo = Geminiチャットそのもの」「Veo = Runwayと同じサービス」という混同が出やすいため、モデル名と提供形態（アプリ/API）を分けて理解するのがポイントです。

Veo 3とは

Veo 3は、Google DeepMindが公開する動画生成モデルファミリーの一つです。テキストプロンプトや参照画像から短尺動画を生成でき、従来の無音動画生成に加えて、会話や効果音を含むネイティブ音声の生成にも対応しています。

一般ユーザーはGeminiアプリやGoogle Flowから、開発者はGemini API（モデルID例：veo-3.0-generate-001）から利用します。後継のVeo 3.1も提供されており、Fast/Standard/Liteなど速度・品質のバリエーションがあります。

Googleエコシステム内の画像生成（Imagen）やチャット（Gemini）と並ぶ「動画生成の柱」として位置づけられ、試験ではGoogle主要AIの一員として名前が出ることがあります。

できること（主な機能）

Text-to-Video

テキストの指示からシーンを生成し、カメラワークや被写体の動きを含む動画を作れます。

Image-to-Video

静止画を起点に動きを付与し、コンセプトアートや商品画像から動画案を起こせます。

ネイティブ音声生成

セリフや環境音を動画と同時に生成し、後から別ツールで音声を足す手間を減らせます。

高解像度出力

720p・1080pに加え、モデルやプランにより4K相当の生成にも対応します（公式仕様はモデルごとに異なります）。

Gemini / Flow 連携

チャットや制作スタジオのフローに組み込み、Googleアカウント基盤で試作から共有まで進められます。

API連携

開発者はGemini API経由でアプリやワークフローに動画生成を組み込めます（Paid Tier）。

よくある誤解

代表的な誤解は「Veo 3 = Geminiチャット」です。Geminiは対話アプリの名称であり、Veo 3はその中で使われる動画生成モデルです。同様に「Googleの生成AI = すべてGemini」と一括りにせず、画像（Imagen）・動画（Veo）・音声など役割で分ける必要があります。

もう1つは「Veo = Runway」の混同です。どちらも動画生成AIですが、VeoはGoogle製モデルでGemini/Flow/APIから提供され、Runwayは独立した制作プラットフォームです。課金もGoogle AIクレジットやAPI秒課金と、Runwayのクレジット制で異なります。

料金の考え方（2026年6月時点）

Veo 3の料金は利用経路によって異なります。開発者向けAPIと一般向けGoogle AIプランを分けて理解してください。以下は公式情報に基づく2026年6月時点の整理です。

Gemini API（開発者向け）

Gemini APIの公式Pricingによると、Veo 3はFree Tierでは利用不可で、Paid Tierの従量課金となります。音声付き動画（デフォルト）の目安は次のとおりです。

モデル 料金（Paid Tier・秒あたりUSD） 備考
Veo 3 Standard（音声付き）$0.40/秒720p/1080p
Veo 3 Fast（音声付き）$0.10（720p）／$0.12（1080p）／$0.30（4K）速度優先

音声なし生成はおおむね半額程度になる場合があります。音声処理の問題で生成に失敗した場合は課金されないことがあります（公式Pricingの注記）。

Google AIプラン（一般向け）

GeminiアプリやGoogle Flowでは、Google AI Plus / Pro / Ultraなどのプランに含まれる月次AIクレジットから消費します。Google Oneヘルプ（2026年6月時点）の例では、Google AI Pro（月1,000クレジット）でVeo 3.1 Fastは1本あたり20クレジット（最大50本程度）、Veo 3.1 Qualityは100クレジット（最大10本程度）といった換算が示されています。

プラン名・月額・付与クレジット・日次上限は更新が速いため、利用前にGoogle Oneの公式ページで最新条件を確認してください。

はじめ方・基本的な使い方

最短で試す流れは次のとおりです。

  1. Googleアカウントを用意 GeminiアプリまたはFlowにサインインします。
  2. プランとクレジットを確認 無料枠かGoogle AIプランかを確認し、動画生成の上限を把握します。
  3. プロンプトを設計 被写体・動き・カメラ・音声（セリフや環境音）を具体的に記述します。
  4. 生成と選別 Fastモデルで試作し、必要に応じてQuality/Standardで本番用を生成します。
  5. API利用時は課金設計 秒課金のため、尺と解像度・音声の有無を事前に決めてコストを管理します。

よくある質問

Veo 3は無料で使えますか？

Gemini APIのVeo 3はFree Tierでは利用できません。一般ユーザーはGoogle AIプランの範囲でGeminiアプリやFlowからアクセスできますが、クレジットや利用上限があります。

Veo 3とGeminiチャットは同じですか？

同じではありません。Geminiは対話型AIアプリの総称で、Veo 3はその中で使われる動画生成モデルです。試験ではサービス名とモデル名を区別して覚えてください。

Veo 3とRunwayの違いは何ですか？

Veo 3はGoogle製モデルでGeminiやFlowと連携します。Runwayは独立した動画生成・編集プラットフォームです。課金体系も異なります。

Veo 3の料金はどう計算されますか？

Gemini APIでは生成秒数に応じて課金されます（例：Standard音声付き $0.40/秒）。一般向けはGoogle AIプランの月次クレジットから消費します。成功した生成のみ課金となる場合があります。

Veo 3.1とVeo 3の関係は？

Veo 3.1は後継世代で、GeminiアプリやFlowではVeo 3.1 Fastなどが提供されています。試験ではGoogle製・音声同期動画生成の位置づけを押さえ、詳細は公式で確認してください。
