試験で問われる見方

動画生成AIの論点では、ツール名の変遷（Dream Machine→Luma）より、動画生成AIの利用上の注意（商用可否・クレジット・出力確認）が問われやすいです。

「Luma＝単一の動画モデル」ではなく、複数モデルを扱うプラットフォームである点と、「アプリのサブスク＝API料金」と混同しない点を押さえておくと得点しやすくなります。

Luma Dream Machine（Luma）とは

Lumaは、動画・画像・音声の生成モデルを一つの制作フローで扱えるクリエイティブアプリです。2024年頃に話題となったDream Machineは動画生成の旧ブランド名で、2026年6月時点では公式にLumaへ統合されています（公式のAI向け情報でもDream Machineは旧称とされています）。

自社の動画モデルRay3に加え、Veo・Klingなど第三者モデルもプラットフォーム上から利用できる構成です。Luma Agentsがプロジェクト単位でモデル選択や生成を補助します。

試験では旧名称「Dream Machine」で資料に載っている場合と、現行の「Luma」表記が混在しうるため、同一サービスの名称変更として理解しておくと混乱しにくくなります。

できること（主な機能）

Text-to-Video / Image-to-Video

テキストや画像から短尺動画を生成。Ray3や連携モデルで解像度・音声の有無を選べます。

Luma Agents

動画・画像・音声モデルを横断して制作を進めるエージェント層。プランにより利用上限が異なります。

Ray3 動画モデル

Luma独自の動画生成モデル。Draft/Quality、解像度ごとにクレジット消費が定義されています。

画像・音声生成

Uni-1、Photon、ElevenLabs連携など、動画以外のメディアも同一アプリから扱えます。

ゲスト共同編集

Plus以上でゲストコラボレーターへの編集アクセスを付与できます（公式Pricing）。

Developer API

platform.lumalabs.ai向けAPIはアプリサブスクとは別のクレジット課金です。

よくある誤解

代表的な誤解は「Luma Dream Machine＝今も別サービス」です。現行公式ブランドはLumaであり、Dream Machineは旧称です。URLや試験テキストに旧名が残っていても、指す対象はLumaプラットフォームです。

もう1つは「アプリに払えばAPIも使い放題」です。消費者向けサブスクリプションとAPIクレジットは別管理です。開発連携ではAPI Pricingを個別に確認する必要があります。

はじめ方・基本的な使い方

  1. アカウント作成app.lumalabs.aiで登録し、トライアルクレジットまたはプランを確認します。
  2. 制作目的を定義動画のみか、画像・音声も含むかでAgentsの使い方が変わります。
  3. モデルを選択Ray3か連携モデル（Veo、Kling等）かを選び、消費クレジットを確認します。
  4. 生成と検品複数案を比較し、商用案件ではライセンス条件を再確認します。
  5. API連携時は別途設定開発者向けはplatform.lumalabs.aiでAPIクレジットを管理します。

よくある質問

Luma Dream MachineとLumaの関係は？

Dream Machineは旧称で、現行はLuma（Luma App）に統合されています。試験では名称変更として理解してください。

Lumaは無料で使えますか？

トライアルクレジットはありますが、恒常的な無料プランは個人向けPricingには明示されていません。継続利用はPlus（$30/月）などが中心です。

LumaとRunwayの違いは何ですか？

Lumaはマルチモデル＋Agents、RunwayはGen-4と編集UIが強みです。料金体系も異なります。

LumaアプリとLuma APIの料金は同じですか？

同じではありません。サブスクリプションとAPIクレジットは別ウォレットです。

Lumaは商用利用できますか？

Plus以上にCommercial useが含まれます。最新の利用規約を確認してください。
