Claude Mythos 5とは？Glasswing限定——分類器なしのMythos本体

読み：クロード・ミトス・ファイブ ／ 英：Claude Mythos 5

更新日：2026年6月11日 読了目安：約6分

Claude Mythos 5は、Anthropicが開発したMythosクラスの最上位LLMで、2026年6月9日よりProject Glasswing承認パートナー限定で提供されています。基本能力はClaude Fable 5と同等ですが、安全性分類器を持たない——高リスク領域でも拒否しない——点が決定的な違いです。本記事は沿革の暗記より、「なぜ一般公開されないのか」の設計思想に焦点を当てます。

試験で問われる見方

試験では「高性能モデル＝誰でも使える」ではなく、提供形態と安全上の理由がセットで問われます。Mythos 5が限定公開なのは、分類器なしの最上位能力が悪用リスクを伴うため——という整理が要点です（TF-499）。

Anthropic＝Claudeの開発元（TF-0142）。Mythosクラスの能力を一般利用したい場合はFable 5を選ぶ——この二段構えは生成AIパスポートでもG検定でも再利用しやすいです。

最も注意が必要な誤解は「Mythos 5はFable 5より高性能」です。差は性能ではなく分類器と公開範囲です。

Claude Mythos 5とは

Claude Mythos 5（APIモデルID：claude-mythos-5）は、MythosクラスのLLMです。Fable 5と基本能力は同等とされますが、安全性分類器を搭載していません。

分類器がないため、サイバーセキュリティ・生物学・化学など、Fable 5では自動拒否される領域のリクエストも処理できます。ソフトウェアの脆弱性発見やセキュリティ研究など、制限の少ない最上位モデルが必要な用途向け——ただし審査済み組織に限定——という位置づけです。

コンテキスト100万トークン・最大出力128k・料金体系（入力$10/出力$50 per 100万トークン）はFable 5と同等です。

Project Glasswingとは

Project Glasswingは、AnthropicがClaude Mythosシリーズへのアクセスを管理する限定プログラムです。利用組織の安全性・利用目的を審査したうえで、Mythosモデルへのアクセスを許可します。

Glasswingに参加できない場合、Anthropicは一般公開のMythosクラスモデルであるClaude Fable 5の利用を推奨しています。試験では「Mythos＝Glasswing限定、一般ユーザー＝Fable」と覚えるとすり替えに強くなります。

よくある質問

Claude Mythos 5はなぜ一般公開されていないのですか？

安全性分類器を持たない最上位モデルであるため、サイバーセキュリティや生物・化学などの高リスク領域で悪用されるリスクが高いとAnthropicが判断しています。そのためProject Glasswingを通じて審査済みの組織のみに限定提供されています。

Claude Mythos 5とClaude Fable 5の能力に差はありますか？

基本的な言語モデルとしての能力は同等とされています。主な違いは安全性分類器の有無と提供対象です。Fable 5は安全性分類器を内蔵した一般公開版、Mythos 5は分類器なしの限定公開版です。

Mythosクラスとはどういう意味ですか？

Anthropicのモデル体系において最上位の能力ティアを指す分類です。複雑な推論・ソフトウェアエンジニアリング・長期エージェント作業で最高レベルの性能を持つとされています。一般向けにはFable 5、限定向けにはMythos 5がこのクラスに属します。
