Claude 3 Haikuとは？俳句の軽さと大量処理の入口

Claude 3 Haikuとは

2024年3月、AnthropicはClaude 3ファミリー——Sonnet・Opus・Haiku——を発表しました。Haikuはそのうち最後に広く提供された軽量帯で、企業のデータパイプラインやカスタマーサポートの一次処理向けに位置づけられました。

Claude 3世代はマルチモーダル（画像入力など）にも対応が進みましたが、Haikuの主戦場は依然テキスト中心の大量・高速タスクです。画像理解まで含めるとコストとレイテンシのトレードオフが変わるため、用途で使い分けます。

詩の名前が示す役割

Haiku（俳句）は短い形式の詩——17音の凝縮——にちなんだ名称です。製品上の意味は「短く速く返す」ことに対応します。

階層名文学でのイメージモデルでの役割
Haiku最短の詩形最軽量・高速・低コスト
Sonnet十四行詩（中編）標準・バランス
Opus大規模な作品最上位・深い推論

Googleの Pro/Flash、OpenAIの GPT-4o/mini と命名は異なるが、「軽量帯＝大量処理」という思想は各社共通です。

大量処理の設計思想

シナリオHaiku向きSonnet/Opus向き
問い合わせの一次分類（1日数千件）◎コスト過多になりがち
商品タグの自動付与◎単純タスクでは過剰
契約書の深い照合・矛盾分析△◎
複雑なコーディング設計△◎（Opus寄り）

二段構成——Haikuでふるい分け → 難件だけSonnetへ——はTF-0428のコスト意識と両立する実務パターンです。

よくある質問

Haikuは何を意味しますか？

俳句（haiku）から取った階層名で、短く軽い形式の詩にちなみ、最軽量・高速のモデル帯を示します。試験では文学の知識より「Haiku＝最軽量、Sonnet＝標準、Opus＝最上位」の役割で覚えます。

Claude 3 HaikuとSonnetの違いは？

同一世代（Claude 3）のサイズ階層差です。Haikuは速度とコストを優先し、Sonnetは性能と速度のバランスを取った標準帯です。難問・深い分析はSonnetやOpus向き、大量の分類・短い要約はHaiku向きと整理します。

Haikuは無料版ですか？

いいえ。Haikuはモデル階層の名称であり、無料プランや旧モデルとは別軸です。APIではトークン課金があり、軽量だから無料という意味ではありません。
